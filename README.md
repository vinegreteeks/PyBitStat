# PyBitStat
CLI-утилита для статистического анализа числовых выборок и побитовых операций с поддержкой экспорта отчётов в TXT/CSV.

## Quick Start
# 1. клонируем репозиторий
git clone https://github.com/vinegreteeks/PyBitStat.git
# 2. переходим в директорию
cd PyBitStat
# 3. запускаем
python pybitstat.py

## Features
- Фильтрация выборки (только чётные).
- Расчет статистики (сумма, мин, макс, среднее).
- Подсчет знаков (плюс, минус, ноль).
- Работа с битами (перевод в binary, установка/снятие битов).
- Экспорт в файлы(txt/csv).
